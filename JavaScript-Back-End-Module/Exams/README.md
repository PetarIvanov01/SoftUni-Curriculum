## Getting Started

To get started with the exam files, follow the steps below.

### Downloading the Exams

You can download the exam files by following these instructions:

1. Go to the [Exams folder](https://github.com/your-username/your-repo/tree/main/Exams) in this repository.
2. Click on the exam file you wish to download.
3. On the exam file page, click the "Download" button or use the "Download" link to save the file to your computer.

### Installing Dependencies

Before running the exam files, make sure to install the required dependencies. Here's how:

1. Open a terminal or command prompt.
2. Navigate to the root directory of the downloaded exam files.
3. Run the following command to install the required dependencies specified in the `package.json` file:

```shell
npm install
